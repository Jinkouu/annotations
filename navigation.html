ANNOTATIONS IN NARRATIVE VISUALIZATION

This is a framework for annotation recommendations for different audiences in narrative visualization. By Jonathan Nguyen

Recommendations

1. Click Anywhere

Screenshot - 1

30 Years of American Anxieies

The Pudding is a website that cater towards a primarily 18-24 year old demographic (Similar Web)

In this example, the annotation is triggered by a popup. This popup can appear through loading the webpage, scrolling to the section containing the popup, or clicking on a prompt that opens the popup. Once the annotation pops up, the user can click anywhere on screen, closing the popup.

Recommendation

Use for a younger audience (less than 24 years old) for intuitive use

Use for long annotations where other navigation buttons would be inconvenient to click

DO NOT use for a senior audience OR use alongside explicit navigation options

2. Swipe

SWIPE TO CLOSE

In this example, the annotation is triggered by a popup. This popup can appear through loading the webpage, scrolling to the section containing the popup, or clicking on a prompt that opens the popup. Once the annotation pops up, the user can swipe the annotation away, closing the popup.

Recommendation

Use for mobile narrative visualisations

DO NOT use for a senior audience OR use alongside explicit navigation options

 

3. Hover

Screenshot - 1

A Disappearing Planet

The Propublica is a website that cater towards all audiences with a senior audience (age 55-64) demographic being the largest demographic. (Similar Web)

In this example, the visualisation utilises a universal help symbol that when hovered over, produces the annotation. It always appears on the right of the symbol. The annotation disappears when the cursor is off the symbol.

Recommendation

Use for a wider audience

Use ONLY for web-based narrative visualisations

MUST use when targeting an audience with reduced motor skills

DO NOT use for a senior audience
