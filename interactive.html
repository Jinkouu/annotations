ANNOTATIONS IN NARRATIVE VISUALIZATION

This is a framework for annotation recommendations for different audiences in narrative visualization. By Jonathan Nguyen

Recommendations

1. Highly Interactive

Screenshot - 1

The CRUTCH Rankings on The Pudding

The Pudding is a website that cater towards a primarily 18-24 year old demographic (Similar Web)

In this example it uses highly interactive rollovers as annotations in narrative visualization (see screenshot - 1). Only when the content is rolled over does the annotation appear. It appears directly above the content item

Recommendation

Use only for 18-24 audience

Use only for information that is not necessary to comprehend information

DO NOT use for audience with low visual literacy
